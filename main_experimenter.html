main_experimenter module

Auxiliary script with command-line user interface for conducting experiments on classifiers related to this project: FastRealBoostBins (two variants) against other ensemble classifiers from sklearn.ensemble.

For help on the script arguments execute:

python main_experimenter.py -h

For usage examples, see the README.md file in project’s repository (https://github.com/pklesk/fast_rboost_bins).

Dependencies

  • numpy, math: required for mathematical computations.

  • numba: required for just-in-time compilation of crucial computational functions and CUDA kernels (decorated by @jit and @cuda.jit imported from numba).

  • sklearn: required for inheritence and other sklearn API purposes,

  • colorama: required CLI purposes.
